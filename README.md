# ItsElijahWood's Windows Neovim config

## Install


Neovim for highest version https://github.com/neovim/neovim
PHP for intelephense https://www.apachefriends.org/download.html
PHP-CS-Fixer for PHP Code Formatting https://github.com/PHP-CS-Fixer/PHP-CS-Fixer
NPM for downloading Mason https://nodejs.org/en/download
FZF for quick swapping projects https://github.com/junegunn/fzf
FD alternative to FIND https://github.com/sharkdp/fd

## Env System Variables (Windows)
    C:\Program Files\PHP-CS-Fixer-<version>
    C:\Program Files\Neovim\bin
    C:\Program Files\xampp\php
    C:\Program Files\nodejs
    C:\Program Files\fzf
    C:\Program Files\fd-<version>-pc-windows-gnu
    C:\Program Files\fzf-<version>-windows_amd64
